α  Type I error — reject true H₀

Distribution of test statistic under H₀ and under true μ′
Power curve vs effect size δ
α — Type I error rate
Probability of rejecting H₀ when H₀ is actually true. Set by the researcher via significance level.
β — Type II error rate
Probability of failing to reject H₀ when it is false. Decreases as n ↑ or effect size ↑.
Power = 1 − β
Probability of correctly detecting a false H₀. Conventional target: 0.80 or 0.90.
